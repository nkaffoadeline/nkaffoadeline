# 🎓 School Management System

## 📌 Description
Application web de gestion scolaire permettant de gérer les étudiants, enseignants et cours.

## 🚀 Technologies
- Angular
- Express.js
- MongoDB

## 🎯 Fonctionnalités
- Gestion des étudiants
- Gestion des enseignants
- Gestion des cours

## ▶️ Lancer le projet
npm install  
npm start
